# Climate change simulator

## Overview
This project is an interactive 3D geospatial web application built using CesiumJS and React.
It allows users to explore a 3D globe, navigate to coordinates, and simulate enviormental and climate-based impacts on selected regions. 

The long-term goal is to build a climate change simulator that can model enviormental changes, as well as natural disasters and what effects these do on the climate over time or in targeted areas.

---

## Core Features (MVP Phase 1)

- Interactive 3D globe using CesiumJS
- Cick-and-drag globe navigation
- Fly-to-location using coordinate input
- Search bar for latitude/longitude navigation
- UI control panel using Bootstrap
- Basic disaster simulation menu (UI only)
- Sliders for enviormental variables (UI only)

---

## Planned Features (Futrue Phases)

- Climate simulation logic (temperature, rainfall, etc.)
- Region-based impact visualization
- Time-based decay simulation (future modeling)
- Backend integration using Django REST API
- Scenario saving system
- PDF report generation of simulation results
